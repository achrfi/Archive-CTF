# Ransom Reborn
**ID:** 25
**Category:** Forensics
**Points:** 500

## Description
A professor fell victim to a phishing link, downloading what they assumed was a "Campus System Update." Instead, a hidden ransomware payload silently encrypted all their vital research files. Incident Response has isolated a sample of the malicious application and a few of the encrypted files. As a forensic analyst, you must reverse the ransomware, try to analyze encryption logic to save the professor's life's work.
