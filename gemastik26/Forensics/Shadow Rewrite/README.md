# Shadow Rewrite
**ID:** 21
**Category:** Forensics
**Points:** 500

## Description
Workstation `Lab TULT 0615, Lab Foresty (Forensics and Security)` disita setelah salah satu akun admin gagal login usai insiden dini hari. Tim IR berhasil mengambil beberapa artefak Linux dari mesin tersebut, tetapi belum bisa memastikan kredensial aktif akun yang terdampak.

Telusuri artefak yang tersedia, tentukan akun yang benar-benar relevan dengan insiden, lalu pulihkan kredensial aktifnya untuk mendapatkan flag dengan format `FORESTY{hostname_password}`.
