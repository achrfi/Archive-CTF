# The Eepy Koala
**Category:** Cryptography
**Points:** 194

## Description
In a peaceful eucalyptus forest, a very sleepy genius koala has just photographed a major secret. Eager to drift off to sleep under the light of the full moon, he created an encryption script for it.

While yawning, he thought, "I don’t need a complicated key. My short-term memory is only enough for 16-bit. I’ll use that number to scramble everything from the broadest key to the position of every single leaf in my photo. I’ve ensured the photo's size fits perfectly into my secret boxes."

Based on the provided file, can you restore it to its original state before the koala wakes up? Remember, time is precious, don't get stuck counting the entire forest if you only need to check the tips of the leaves.
